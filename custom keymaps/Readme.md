# Custom Keymappings

The Plum 84 can be customised using the KB84 software.  My current custom mappings are saved in `my_customer_mapping.pro`.

## my_customer_mapping.pro


| Original Key   | Mapping   |
|----------------|-----------|
| Caps Lock      | Escape    |
| Fn + Caps Lock | Caps Lock |
| Right Ctrl     | Insert    |
